# Changelog

All notable changes to this project will be tracked here.

The version format is `MAJOR.MINOR.PATCH`, mapped in this repo as:

- `Proud Version` = major
- `Stable Version` = minor
- `Patch Number` = patch

## [0.1.0] - 2026-03-26

Initial stable tracked baseline.

- established the Godot 4 project structure, gameplay scene, and test harness
- added mobile-first controls, camera swap behavior, and retro HUD styling
- added deterministic visual capture tooling and repo-local Codex workflow docs
- added Web export and HTTPS phone-testing helpers
